# DevContainer (OMBbApi + Vue)

### Vue 프로젝트 실행

```
cd vue-app

yarn serve --port 80
```

### API Key 등록하기

###### env.js (DevMovieVue/vue-app/src/env.js)

```
export default {
  apikey: 'User API Key'  << 발급 받은 키를 넣어 주세요.
  ...
}
```
